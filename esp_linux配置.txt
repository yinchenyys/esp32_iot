=======================                                  lvgl                        ==============================================
1.创建一个lvgl_display文件夹
idf.py create-project lvgl_display

2.创建components

3.在lvgl_display文件夹的make文件写入 
set(EXRA_COMPONENT_DIRS ./components) （弹幕说最好加在include前）

4.在componrnts进行克隆
git clone --recursive https://github.com/lvgl/lvgl

5.git tag 查看版本

6.切换到 v8.3.10

7.创建bsp放自己的组件，把~/esp/esp32-board/display/main的
cp cst816t_driver.* ~/esp/lvgl_display/components/bsp/
cp st7789_driver.* ~/esp/lvgl_display/components/bsp/
 显示屏文件移到 ~/esp/lvgl_display/components/bsp/

8.并在bap创建CMakeList.txt
写入
idf_components_register(
    SRCS "cst816t_driver.c"  "st7789_driver.c"  
    INCLUDE_DIRS "."
    REQUIRES esp_lcd
)
其中
SRCS 放源码  
INCLUDE_DIRS 包含路径
REQUIRES  组件依赖项（main文件默认对所有组件依赖）

个人编译错误总结：
(1).一直显示找不到 #include "st7789_driver.h"  和  #include "cst816t_driver.h"
	CMakeLists.txt                                                                  编写错误    CMakeLits   (E)->(Y)    CMakeLists
	set(EXTRA_COMPONENT_DIRS ./components)                编写错误    EXTA   (E)->(Y)    EXTRA
(2).heap_caps_calloc缺少参数
	应该用heap_caps_malloc
(3)很多错误，原因
	#include "freertos/FreeRTOS.h"
	#include "freertos/task.h"               要放在"freertos/FreeRTOS.h"之后
